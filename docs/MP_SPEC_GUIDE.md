# mp_spec.py 활용 가이드 (검토 초안)

`mp_spec.py`는 Multipass 인스턴스 스펙 정보를 파싱해 자동화에 활용할 수 있도록 설계된 스크립트입니다. 현재는 직접 호출되는 경로가 없으며, 향후 Python 기반 유틸/오케스트레이터로 확장하는 것을 전제로 합니다.

## 사용 목적(예시)
- VM 리소스 확인 자동화(메모리/CPU/디스크)
- 테스트 시나리오별 프로파일링
- 리소스 스펙에 따른 클러스터 재생성 정책 판단

## 향후 Python 통합 방향 제안
- `scripts/py/` 디렉터리 신설 후 Python CLI 유틸(예: `k8s_tool.py`)로 통합
- `mp_spec.py`를 내부 모듈로 재사용하거나 `mp_spec.py` 기능을 Python 패키지로 흡수
- `scripts/k8s-tool.sh`에서 Python CLI를 호출하는 옵션 제공

## 현시점 결론
- 현재는 Bash 기반 오케스트레이션을 유지하되, 추후 기능 확장(테스트/메트릭) 시 Python 전환을 고려
