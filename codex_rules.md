# codex_rules.md

## 맥락 유지
- 매 작업 시작 전 docs/PROGRESS_LOG.md를 읽고 현재 진행 중인 Phase를 확인한다.

## 작업 완료 의무
- 모든 작업 완료 후 반드시 [Sync Report]를 작성한다.
- GitHub 업로드용 Git 명령어를 생성해 제공한다.

## Operational Loop
- [Context Check] docs/PROGRESS_LOG.md 확인
- [Rule Compliance] rule.md 및 본 문서 준수
- [Log Update] 작업 종료 후 docs/PROGRESS_LOG.md 업데이트
- [Sync Report] 지정된 양식 출력
